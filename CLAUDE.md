# CLAUDE.md

## Project: bmad-pokedex
Pokemon discovery website using PokeAPI.co

## Git Commands
Use simple git commands without `-C` flag. The working directory is already the project root.
- Use `git status` not `git -C /path/to/project status`
- Use `git diff` not `git -C /path/to/project diff`
- Commit after completing each BMAD workflow (story, code review, etc.)

## BMAD
Commands: `/bmad:bmm:workflows:<workflow-name>`

Status: `docs/bmm-workflow-status.yaml`
